# Fit-for-Purpose Architecture

This repository demonstrates different architectural approaches for the same problem domain: a URL Shortener.

## Variants

### 01 - Minimal API

A pragmatic MVP implementation using .NET 10 Minimal APIs and in-memory storage.

Focus:
- Fast delivery
- Minimal complexity
- Clear, readable code

See `01-minimal-api/` for details.
